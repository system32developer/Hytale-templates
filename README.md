# This is a template system for Hytale Plugins (supports java and kotlin)
This project allows to create simple plugins using Hytale API with one click, it creates the Main class, Commands class, Events class, and also adds a task to your Intellij that will create and start a Hytale Server with your plugin on it

## Tutorial:

1. Go to Intellij & click on Plugins is on the left bar

![First step](https://cdn.discordapp.com/attachments/1415912570509856870/1461741164636934389/image.png?ex=696ba80b&is=696a568b&hm=9e61dc8edf371d5c8e084a6f8414d325667e7d226ec9dd3e1709edf92f62aa07&)

2. Search Minecraft Development on Marketplace (This plugins is used to dev Minecraft plugins but it has a Template System and we will use our Hytale Template there)

![Second Step](https://media.discordapp.net/attachments/1415912570509856870/1461741689122328606/image.png?ex=696ba888&is=696a5708&hm=376731eaaa3835855b23a84c916769951d11a8a96317dd7eada36eb17c9ff872&=&format=webp&quality=lossless)

3. Once you downloaded it will probably ask to restart Intellij, then open it again, Click on the 3 dots at the bottom left and select Settings, then go to Languages & Frameworks Section 

![Third Step](https://media.discordapp.net/attachments/1415912570509856870/1461742070854189140/image.png?ex=696ba8e3&is=696a5763&hm=e7e943bf6b6cc3204ff2eb556a6bb665cd83e2d7f74492773df4fb325e247a15&=&format=webp&quality=lossless)

4. On that section go for Minecraft Development, you don't need to expand it, just click on it, you will see this

<img width="982" height="728" alt="image" src="https://github.com/user-attachments/assets/85fc9384-b2e1-4437-b022-483637f8f26e" />

5. Go to Creator, Templates repositories, its bellow then click on the +, name it Hytale and select local on the left

<img width="358" height="299" alt="image" src="https://github.com/user-attachments/assets/70cc7ee6-85c1-4764-b1b0-e951fd795b5c" />

6. Click on the pencil, it's the edit button, and now you need to download and unzip the code on the repo, save the files on a folder somewhere, then select the folder on here, click Apply and OK

<img width="540" height="330" alt="image" src="https://github.com/user-attachments/assets/0d6ebcc5-e425-49d7-bbfe-155620b95351" />

7. Now when you try to create a project Select Hytale on the options

<img width="1063" height="924" alt="image" src="https://github.com/user-attachments/assets/328100d3-704a-438e-bd5a-54c02d3eaf67" />

8. It only Supports Java And Kotlin (Both with Gradle, no Maven supported), please select JDK 25

<img width="778" height="845" alt="image" src="https://github.com/user-attachments/assets/ccb54cff-0c1e-40b9-95f9-498637c430c5" />

9. That's all, everytime you create a new project you will have few classes and server setup! if you had an issue please create one
